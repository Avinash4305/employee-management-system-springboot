# Employee Management System

## Overview
A Spring Boot MVC CRUD application for managing employee records.

## Technologies Used
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL
- JSP

## Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee
- Salary Calculations
- Exception Handling

## Architecture
Controller → Service → Repository → Database

## Tools Used
- Eclipse IDE
- MySQL
- Maven

## Run the Project
1. Configure MySQL database
2. Update application.properties
3. Run Spring Boot application
4. Open: http://localhost:4041/MiniProjectApp/
